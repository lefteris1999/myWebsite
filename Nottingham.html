Bachelor's Degree in Computer Science

University of Nottingham (2019 - 2022)

Course Overview

This degree provided a solid foundation in Computer Science, emphasizing critical areas like algorithms, data structures, software development, and Object-Oriented and Functional Programming.

Notable Modules:

  • Year 1:
    • Mathematics for Computer Science
    • Programming and Algorithms
    • Systems and Architecture
    • Databases and Interfaces
    • Fundamentals of Artificial Intelligence
    • Programming Paradigms
  • Year 2:
    • Software Engineering Group Project
    • Algorithms Correctness and Efficiency
    • Operating Systems and Concurrency
    • Developing Maintainable Software
    • C++ Programming
  • Year 3:
    • Collaboration and Communication Technologies
    • Mobile Device Programming
    • Professional Ethics in Computing
    • Human-AI Interaction
    • Computer Security
    • Fundamentals of Information Visualisation
    • Malware Analysis
