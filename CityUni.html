Master's Degree in Data Science

City, University of London (2022 - 2023)

Course Overview

This degree provided a solid foundation in Data Science, emphasizing critical areas like exploratory data analysis, data visualisations, machine learning/deep learning and artificial intelligence.

  • Big Data
  • Information Retrieval
  • Machine Learning
  • Natural Language Processing
  • Neural Computing
  • Principles of Data Science
  • Visual Analytics

Key Projects

Aspect Based Sentiment Analysis

Developed a tool that analyzes customer reviews to classify sentiment using Natural Language Processing (NLP) techniques. Python, Scikit-learn, and NLTK libraries were used.
